Errores de validación en el alta del club de orientación

El alta de un club de orientación tiene las siguientes reglas de validación:

  • El nombre del club es obligatorio
  • El nombre del club debe tener al menos 3 caracteres


Si no se cumple alguna de estas reglas de validación, el sistema debe informar al usuario qué campo y por qué no cumple la validación.

Error de validación al no dar valor al campo nombre del club

Dado un usuario, a través del menú pulsa sobre el link de gestión de clubs y accede a la página de Listado de clubs de orientación,
pulsa sobre el botón Nuevo club y el sistema dirige al usuario a la página de Alta de clubs de orientación,
rellena el formulario de alta de un club excepto el campo nombre del club,
pulsa sobre el botón de Crear club y
el sistema mantiene al usuario en la página de Alta de clubs de orientación e informa que El nombre del club es obligatorio.
