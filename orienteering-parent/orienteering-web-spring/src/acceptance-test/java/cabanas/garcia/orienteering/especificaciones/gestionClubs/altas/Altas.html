Funcionalidad del alta de clubs de orientación

Se requiere que un usuario pueda dar de alta un club de orientación a través de un formulario web.

El usuario rellenará la información del club que crea conveniente y podrá dar de alta al club mediante un botón/link.

En el caso que el proceso de alta tenga éxito, el sistema redirigirá al usuario al listado de clubs y mostrará al usuario un mensaje informativo indicando que la operación se realizó con éxito.

En el caso que el proceso de alta produzca errores, éstos serán debidamente informados al usuario.

También se le proporcionará al usuario un botón/link que permitirá al usuario cancelar la operación de alta del club, y el sistema redirigirá al usuario al listado de clubs, indicándole con un mensaje informativo que la operación fue cancelada.

Alta de un club de orientación

Dado un usuario, que pulsando sobre el link de gestión de clubs del menú, accede a la página de Listado de clubs de orientación, pulsa sobre el botón Nuevo club, y el sistema dirige al usuario a la página de Alta de clubs de orientación.

El usuario rellena el formulario de alta con la siguiente información:
CampoValor
NombreClub Rumbo

Cuando pulsa sobre el botón de Crear club y
el sistema redirige al usuario a la página de Listado de clubs de orientación e informa que El registro se ha dado de alta correctamente.

Cancelación del alta de un club de orientación

Dado un usuario, que accede al formulario de alta de un club a través del menú, pulsando sobre el link de gestión de clubs, accediendo a la página de Listado de clubs de orientación,
pulsa sobre el botón Nuevo club y el sistema dirige al usuario a la página de Alta de clubs de orientación,
pulsa sobre el botón de Cancelar y
el sistema redirige al usuario a la página de Listado de clubs de orientación e informa que Se ha cancelado la operación.
