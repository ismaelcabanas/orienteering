Funcionalidad de la modificación de clubs de orientación

Se requiere que un usuario pueda modificar los datos de un club de orientación dado de alta en el sistema a través de un formulario web.

El usuario podrá seleccionar el club que desea modificar, accederá a un formulario editable con los datos del club, modificará la información del club que crea conveniente, y podrá actualizar la nueva información del club mediante un botón/link.

En el caso que el proceso de actualización del club tenga éxito, se mostrará al usuario un mensaje informativo indicando que la operación se realizó con éxito y el sistema redirigirá al usuario al listado de clubs.

En el caso que el proceso de actualización produzca errores, éstos serán debidamente informados al usuario.

También se le proporcionará al usuario un botón/link que permitirá al usuario cancelar la operación de actualización del club, y el sistema redirigirá al usuario al listado de clubs, indicándole con un mensaje informativo que la operación fue cancelada.

Actualización de un club de orientación

Dado un usuario que selecciona el club Rumbo Madrid Sanse para modificar sus datos, el sistema dirige al usuario a la página de Edición de clubs de orientación en la que aparece los siguientes datos del club:
Campo Valor
Nombre Rumbo Madrid Sanse

si modifica el nombre del club a Madrid Rumbo Sanse y pulsa sobre el botón de Actualizar

el sistema redirige al usuario a la página de Listado de clubs de orientación e informa que El registro se ha actualizado correctamente.

Cancelación de la actualización de un club de orientación

Dado un usuario que selecciona el club Rumbo Madrid Sanse para modificar sus datos, el sistema dirige al usuario a la página de Edición de clubs de orientación en la que aparece el campo Nombre con el valor Rumbo Madrid Sanse.

si, después de modificar algún dato, pulsa sobre el botón de Cancelar

el sistema redirige al usuario a la página de Listado de clubs de orientación e informa que Se ha cancelado la operación.
