Funcionalidad del detalle de clubs de orientación

Se requiere que un usuario pueda ver los datos de un club de orientación.

El usuario utilizará el formulario de búsqueda de clubs de orientación para buscar el club de orientación que desea ver. Si encuentra el club de orientación, accederá a una página de detalle donde se le mostrará los datos del club.

También se le proporcionará al usuario un botón/link que permitirá al usuario volver al listado de clubs de orientación.
