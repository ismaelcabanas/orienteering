Funcionalidad de consulta de clubs de orientación

Se requiere que un usuario pueda consultar clubs de orientación aplicando unos criterios de búsqueda.

El usuario podrá buscar clubs de orientación por los siguientes criterios de búsqueda:

  • Nombre, la búsqueda por nombre de club devolvería cualquier club cuyo nombre contenga el valor introducido; no se hará distinción entre mayúsculas y minúsculas.

Una vez rellenado los criterios de búsqueda, el usuario pulsará sobre el botón/link Buscar y el sistema mostrará un listado con los resultados de la búsqueda.

En el caso que la búsqueda no devuelva datos, se mostrará al usuario un mensaje informativo indicando que el resultado de la búsqueda no devolvió ningún resultado.

También se le proporcionará al usuario la funcionalidad de inicializar el criterio de búsqueda mediante el botón/link Limpiar.

Consulta de clubs de orientación con resultados por nombre de club

Dado un usuario que, pulsando sobre el link de gestión de clubs del menú, accede a la página de Listado de clubs de orientación.

El usuario rellena el formulario de búsqueda con los siguientes datos:
Campo de búsqueda Valor
Nombre consulta
y pulsa sobre el botón/link Buscar.

El sistema devuelve como resultado el siguiente listado de clubs que coinciden con los criterios de búsqueda:
Nombre
Consulta Testing Club
y permanece en la página Listado de clubs de orientación.
